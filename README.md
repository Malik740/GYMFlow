# GymFlow - Gym Management System

GymFlow is a desktop-based gym management system developed using VB.NET and SQL Server.  
The system is designed to help gym employees manage daily operations efficiently through a simple and modern interface.  
[CI](https://github.com/Malik740/GYMFlow/actions/runs/28240684064)
---

## Features

- Employee Attendance Management
- Member Registration
- Payment Management
- Subscription Plans
- Member Data Management
- Search and Filter Functions
- Database Integration with SQL Server
- Simple and User-Friendly Interface

---

## Technologies Used

- VB.NET
- Windows Forms
- SQL Server

---

## System Purpose

The purpose of this system is to simplify gym administration tasks and improve employee workflow by providing an organized management system for members, payments, and attendance.
